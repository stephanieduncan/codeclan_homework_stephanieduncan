  1. We can be more quantitative in comparing the distribution of top brand ratings with a normal distribution. You calculated the mean() and sd() of your distribution in an earlier question. Use these values to calculate the proportions of ratings within one-, two- and three standard deviations of the mean. Compare these proportions with those you expect for a normal distribution.
  1. Finding the proportion of ratings within one standard deviation of the mean
samsung_ratings <- mobile_phones %>% 
  select(brand, rating) %>% 
   filter(brand == "Samsung")

mobile_phones %>%
  filter(brand == "Samsung") %>% 
  filter(rating >= samsung_stats$mean - samsung_stats$sd) %>%
  filter(rating <= samsung_stats$mean + samsung_stats$sd) %>%
  summarise(prop_within_1sd = n() / nrow(samsung_ratings))
  1. Within 2 standard deviations
mobile_phones %>%
  filter(brand == "Samsung") %>% 
  filter(rating >= samsung_stats$mean - 2* samsung_stats$sd) %>%
  filter(rating <= samsung_stats$mean + 2* samsung_stats$sd) %>%
  summarise(prop_within_2sd = n() / nrow(samsung_ratings))

Within 3 standard deviations

mobile_phones %>%
  filter(brand == "Samsung") %>% 
  filter(rating >= samsung_stats$mean - 3* samsung_stats$sd) %>%
  filter(rating <= samsung_stats$mean + 3* samsung_stats$sd) %>%
  summarise(prop_within_3sd = n() / nrow(samsung_ratings))
